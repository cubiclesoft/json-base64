When publishing for Chrome Web Store, use the following information for various fields.

Short description:  Import and export files in the JSON-Base64 (.jb64) file format.
Add-on Type:  Sheets
Version:  Latest version as defined by File -> Versions...
Help URL:  https://jb64.org/jb64_for_google_sheets/
Report Issue URL:  https://jb64.org/jb64_for_google_sheets/

Post-install tip:

To import a JSON-Base64 (.jb64) file, select 'Import...' and use the 'Select file' button.

To export to JSON-Base64, select 'Export current sheet...', map the desired output data types for each column, and use the 'Export' button.

*********************** Javascript JSON-Base64 Library ***********************

This directory contains an open source implementation of JSON-Base64 in
Javascript and the relevant test suite.

CryptoJS is under the MIT license.

This implementation is confirmed to pass all tests in the test suite under the
following web browsers:

  - Microsoft Internet Explorer 5.5+
  - Google Chrome 7.0+
  - Mozilla Firefox 4.0+
  - Opera 11.61+
  - Epiphany 3.6.1+
  - Flock 3.5.2+
  - Iceape 2.0.11+
  - Iceweasel 3.5.16+
  - Konqueror 4.11+
  - Luakit 1.6.3+
  - Midori 0.4+
  - Rekonq 2.3+
  - Safari 5.1.7+
  - SeaMonkey 2.1+

Be sure to run the test suite in all browsers that you intend to support.  All
tests should [PASS] and the square in the corner turn green.
